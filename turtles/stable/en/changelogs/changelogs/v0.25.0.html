Description

This version of Rancher Turtles is installed by default as part of Rancher 2.13 and does not require a separate installation. If you are upgrading from earlier versions of Rancher Turtles please refer to the migration guide. Users must ensure they upgrade to Turtles 0.24.3 before upgrading to Rancher 2.13.

With this release of Turtles, the CAPRKE2 and CAAPF providers are no longer installed and managed by the Turtles chart. However, these CAPIProviders will not be uninstalled and will continue to run post-migration.
