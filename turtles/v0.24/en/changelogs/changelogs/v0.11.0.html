Description

üöÄ This release brings two new providers to the list of certified providers: CAPV (vSphere) from experimental to GA and CAPZ (Azure). Moreover, our test suite is now available for prospective end-users to use and add their providers to the list of certified providers. For more information please read here.

Rancher Turtles is an extension to Rancher that brings full Cluster API integration to Rancher.
